Inseting:
---------
"o" -> change into insert mode into below line
"O" -> change into insert mode into above line
"P" -> paste to the above line
"a" -> change into insert mode with one character after
"A" -> change into insert mode with end of line
"I" -> change into insert mode with beginning of line

Searching
---------
"/" -> start searching with "word"
"n" -> next searching word
"N" -> previous searching word
"\*" -> search the word where the cursor is. Next occurence.
"\# -> search the word where the cursor is but the reverse order. Previous occurence.
